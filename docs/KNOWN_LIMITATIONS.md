# 已知限制（MVP 0.1 / M6）

诚实列出当前边界，避免把“未做”包装成故障。

## 产品

- 无 Steam 登录、好友关系与游戏库交集。
- 无移动端、推送与组队社交。
- 无用户生成评论/社区帖。
- 自然语言 AI 默认关闭；无 Key 时返回确定性 `ai_status=fallback`。
- 四分区在真实数据不足时允许少于 20 条，并如实展示空态/早期数据。

## 数据

- 商店搜索多人分类是易变适配器，不构成深度联机画像。
- 熟人联机高影响字段（自建服、私人房、服务依赖）依赖黄金集与人工校正。
- 典型局时长覆盖仍低，未知时显示未知。
- 价格区域默认 `CN/schinese`；历史 US/USD 快照不代表中国区完成。
- 日历/Demo 依赖易变商店字段 + 人工回退。

## 技术

- 单主 SQLite，无 active-active、无网络文件系统部署。
- 服务端默认 loopback；HTTPS 由外部反向代理承担。
- Windows 服务通过 WinSW 托管控制台进程，而非进程内 SCM 实现。
- 桌面自动更新与代码签名策略已文档化，**未**在默认 CI 启用私钥签名（需负责人授权与证书）。
- macOS 以原生 runner 构建/APP smoke 为证据，不跑 `tauri-driver` GUI E2E。
- 性能门槛以本地集成测试与 CI 为目标硬件；生产压测需单独环境。

## 合规

- 隐私政策见 [PRIVACY.md](PRIVACY.md)；遥测默认最小且可关闭（当前 MVP 不以第三方分析 SDK 为依赖）。
- 第三方许可清单由脚本生成，UNKNOWN 行须在商店分发前清理。
- Steam 品牌与素材使用见 [STEAM_BRAND_REVIEW.md](STEAM_BRAND_REVIEW.md)；未获批不得使用受保护徽标。

## 运维

- 备份是运维责任；未配置定时任务时不会自动异地备份。
- 限流默认按设备/会话与 IP；错误开启 `MPGS_TRUST_PROXY_HEADERS` 会导致限流键被伪造。
